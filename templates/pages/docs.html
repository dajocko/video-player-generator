About the application

Video Player Generator is a client side application made with Angular JS and Google's material design. It is developed in HTML, CSS and JavaScript. This application let users develop their own HTML5 video player with custom controls which look and behave same in all major web browsers. Users can select the controls they want and application creates dynamic code as per their selection.

How to create your own video player

Select the controls you want in your video player

See the live preview of your video player

Copy your code

HTML is for markup, CSS is for styling and JavaScript is for behaviour

Add your video's source to video tag in HTML code

If you haven't selected quality change control for your video player, you will get following code, type the source of video in src attribute where id="low"and keep the src and type attribute blank where id="high" enter type of video in type attribute. An example is given below:

<video id="avideo">
  <source id="low" src="yourvideo.mp4" type="video/mp4" />
  <source id="high" src="" type="" />
</video>

If you did select quality control, then you will have following code, type the name of low quality video in source where id="low" and type the high quality video source where id="high". Example is given below:

<video id="avideo">
  <source id="low" src="lowQuality.mp4" type="video/mp4" />
  <source id="high" src="highQuality.mp4" type="video/mp4" />
</video>

Don't forget to link your external CSS and JavaScript files to your html page. href and src attributes should have same name as your files.
Keep both stylesheets in <head> tag of your HTML page
This video player also uses material icons library from google (second style sheet)

<link href="video-player.css" rel="stylesheet" />
<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">

Keep your script file just before the end of body tag

<script src="video-app.js" type="text/javascript"></script>

Other Features

Keyboard

  • Space bar to play/pause the video
  • Enter Key to enter or exit the full screen

Mouse

  • Controls hide after three seconds of inactivity both in full screen and normal mode
  • Controls come back when mouse moves over video
  • Controls hide when mouse leaves the video in mormal size
